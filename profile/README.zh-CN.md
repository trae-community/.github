## 🚀 快速开始

<table>
<tr>
<td width="50%">

### 👀 探索者
```bash
git clone https://github.com/trae-community/trae-demos
cd trae-demos/getting-started
pip install -r requirements.txt
python demo.py
```

</td>
<td width="50%">

### 📖 学习者
1. 浏览 [`trae-learning`](https://github.com/trae-community/trae-learning)
2. 选择你的学习路径
3. 构建实践项目
4. 加入 [`discussions`](https://github.com/trae-community/discussions)

</td>
</tr>
<tr>
<td width="50%">

### 🔨 建设者
1. 从 [`trae-co-creation`](https://github.com/trae-community/trae-co-creation-projects) 选择项目
2. 阅读 `CONTRIBUTING.md`
3. 提交你的第一个 PR

</td>
<td width="50%">

### 💡 创新者
1. Fork [`templates`](https://github.com/trae-community/templates)
2. 集成 [`trae-skills`](https://github.com/trae-community/trae-skills)
3. 与社区分享

</td>
</tr>
</table>

## 🌟 如何参与

<div align="center">

> **"没有单一的入口。TRAE 是一个网络，而不是漏斗。"**

</div>

<details>
<summary><b>🔍 观察者模式</b> — 浏览、观看、被动学习</summary>
<br>

- 浏览仓库并阅读代码
- 观看演示并探索案例研究
- 在讨论中潜水并向他人学习

</details>

<details>
<summary><b>🎓 学习者模式</b> — 跟随路径，构建项目</summary>
<br>

- 遵循结构化的学习路径
- 完成实践项目
- 提问并获得反馈

</details>

<details>
<summary><b>🛠️ 建设者模式</b> — 为活跃项目做贡献</summary>
<br>

- 选择一个活跃的项目并贡献
- 修复 bug、添加功能、改进文档
- 与其他成员协作

</details>

<details>
<summary><b>🚀 先锋模式</b> — 领导和创新</summary>
<br>

- 启动新的共创项目
- 提出实验性想法
- 领导社区倡议

</details>
